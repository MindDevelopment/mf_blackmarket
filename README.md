# mf_blackmarket

A lightweight and configurable black market resource for **FiveM** built for the **Qbox Framework**.

## Features

- Clean and modular codebase
- Black market shop
- Configurable items and prices
- Qbox compatible
- Uses the Ox ecosystem (ox_lib, ox_inventory, ox_target)
- Easy to extend

## Requirements

- Qbox Framework
- ox_lib
- ox_inventory
- ox_target

## Installation

1. Place `mf_blackmarket` inside your `resources` folder.
2. Ensure the required dependencies are installed.
3. Add the resource to your server.cfg.
4. Configure the resource to fit your server.

## Configuration

Most behavior can be configured through the resource configuration files, allowing you to change locations, items, prices and other settings without modifying the core logic.

## Planned Improvements

This repository represents the current black market implementation. A future major version is planned that expands the resource into a complete criminal progression system with missions, levels, reputation and multiple dealer locations.
